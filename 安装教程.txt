【一、插件简介】

网盘通是一款 Chrome 浏览器插件，可以把各家网盘账号集中管理，
点击图标即可一键自动登录，免去每次手动输入账号密码的麻烦。

主要功能：
  - 网盘账号集中管理（支持百度网盘、阿里云盘、夸克网盘等）
  - 一键自动登录，自动打开网盘页面并填充账号密码
  - 密码 AES-GCM 加密存储，安全可靠
  - 支持同一网盘多账号绑定（最多 20 个）
  - 多端数据云同步（通过 Chrome Sync）
  - 备忘录功能（2000 字符）
  - 登录设备管理（最多 5 台）


【二、安装方法（任选其一）】

------------------------------------------------------
方法一：拖拽安装（推荐，最简单）
------------------------------------------------------

1. 打开 Google Chrome 浏览器
2. 在地址栏输入：chrome://extensions  并按回车
3. 打开右上角的「开发者模式」开关
4. 将「网盘通_v1.0.0.crx」文件直接拖入浏览器窗口
5. 弹出确认框，点击「添加扩展程序」
6. 安装完成！浏览器右上角工具栏会出现网盘通图标

------------------------------------------------------
方法二：加载已解压的扩展程序
------------------------------------------------------

1. 将「网盘通_v1.0.0.crx」文件后缀改为 .zip
2. 解压 zip 文件到一个文件夹（如：D:\网盘通\）
3. 打开 Chrome 浏览器，地址栏输入：chrome://extensions
4. 打开右上角的「开发者模式」开关
5. 点击左上角「加载已解压的扩展程序」
6. 选择刚才解压的文件夹，点击确定
7. 安装完成！

------------------------------------------------------
方法三：从 GitHub 源码安装（开发者适用）
------------------------------------------------------

1. 访问项目地址：https://github.com/xwt027/wangpant
2. 下载代码（点击 Code -> Download ZIP，或 git clone）
3. 解压到任意文件夹
4. Chrome 打开：chrome://extensions
5. 开启「开发者模式」
6. 点击「加载已解压的扩展程序」，选择解压后的文件夹


【三、首次使用设置】

1. 点击浏览器右上角的网盘通图标
2. 如果没有添加账好，会弹出引导按钮「+ 添加网盘账号」
3. 点击后进入设置页面，完成以下操作：

   (1) 个人资料：上传头像、设置昵称、邮箱、手机号
   (2) 账号管理：添加网盘账号
       - 选择网盘平台
       - 输入账号（邮箱/手机号）
       - 输入密码（加密存储）
       - 可填写备注区分多个账号
   (3) 备忘录：可记录 2000 字符以内的文字
   (4) 登录设备：查看已登录设备，支持远程移除

4. 设置完成后，点击插件图标即可看到网盘列表
5. 点击任意网盘图标，自动打开登录页面并填充账号密码


【四、常见问题】

Q: 安装时提示「无法从该网站添加应用、扩展程序和用户脚本」？
A: 确保已开启「开发者模式」（chrome://extensions 页面右上角）。

Q: 拖入 crx 文件没有反应？
A: 请尝试「方法二」，将 crx 改名为 zip 后解压加载。

Q: 图标显示异常或为空白？
A: 在 chrome://extensions 页面找到网盘通，点击刷新按钮重新加载。

Q: 密码安全吗？
A: 密码使用 AES-GCM 加密，仅存储在浏览器本地，不会上传到任何服务器。

Q: 如何同步数据到其他电脑？
A: 使用同一 Google 账号登录 Chrome 浏览器，数据会自动同步。


【五、卸载方法】

1. Chrome 打开：chrome://extensions
2. 找到「网盘通」扩展程序
3. 点击「移除」按钮
